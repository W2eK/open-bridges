## Установка и запуск
1. Копирование проекта
    ```shell
    git clone https://github.com/w2ek/open-bridges.git
    ```
2. Установка зависимостей
    ```shell
    cd open-bridges
    yarn install
    ```
3. Создание и запуск билда на `http://localhost:9000/`
    ```shell
    yarn build
    yarn serve
    ```
4. Запуск dev-сервера на `http://localhost:8000/`
    ```shell
    yarn develop
    ```
